# API (Application Programming Interface):
   - Định nghĩa: là một tập hợp các commands, functions, protocols, objects,...
		             giúp hai phần mềm/ứng dụng có thể tương tác và trao đổi dữ liệu
                 qua lại được với nhau mà không cần quan tâm
                 Application được xây dựng, hiện thực như thế nào

# UI (User Interface):
   - Định nghĩa: giao diện giúp cho user tương tác với Application
	             	 mà không cần quan tâm Application được xây dựng, hiện thực
                 như thế nào
